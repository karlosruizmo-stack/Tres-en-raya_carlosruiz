# Tres-en-raya_carlosruiz
- Juego del 3 en raya como un menu multiopcion.
- El juego consiste en la union de tres puntos por parte de los jugadores,si se logra de cualquier forma tanto como vertical horizontal o vertical el jugador ganara un punto.
- El programa tiene una cantidad predefinida de partidas pero si el juegador lo desea puede establcer el modo torneo para realizar un numero mayor de partidas.
- Dentro del programa tambien se ha incluido un mayor numero en la cantidad de union de puntos hasta 4.
